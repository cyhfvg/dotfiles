九、使用建议

  1. 主要工作流：
     使用 Alt + H/J/K/L 在分屏之间移动，使用 Ctrl + Shift + H/J/K/L 调整分屏大小。

  2. 标签页工作流：
     使用 Alt + 数字直接跳转标签页，使用 Alt + Shift + 1/2/3 快速打开常用 Shell。

  3. 安全粘贴：
     多行和大块粘贴警告已开启，适合从网页、AI 回复或文档复制命令时降低误执行风险。

  4. URL：
     URL 检测已开启，终端中的链接可以用 Ctrl + 点击打开。

  5. 透明度：
     已关闭 Ctrl + Shift + 鼠标滚轮改变透明度，避免误操作；保留 Ctrl + 鼠标滚轮缩放字体。
